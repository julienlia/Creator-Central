Adobe Photoshop CC 2017

Before you can do anything, you will need to have Adobe Photoshop installed. You can do this by using this link:

http://www.adobe.com/au/products/photoshop.html

Let's begin with the basics of Photoshop. The Adobe app 'Photoshop' allows you to edit any photo you upload using very extensive and professional tools. When it comes to photo-editing, there is nothing better than using Adobe Photoshop!

As you can see from the image below, there are plenty of tool available to use when editing an image. Under that, you can find a list of what each tool category does:

Selection Tools: These tools are based around selecting a certain area on an image for use with another tool.

Crop and Slice Tools: These tools are based around removing areas of an image entirely.

Measuring Tools: These tools are based around finding the specifics of an image, such as the colour and size of an area on the photo.

Retouching Tools: These tools are based around morphing and manipulating the image without adding something external.

Painting Tools: These tools are based around, as you would expect, creating custom images using in-app tools such as the pencil tool or the brush tool.

Drawing and Type Tools: These tools are similar to the painting tools, however they are more based on text, as can be seen through the pen and type tools.

Navigation Tools: These tools are based around changing your view of an image, through zooming in or moving it to a selected side.
